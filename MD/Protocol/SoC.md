### SoC란?
* System on a Chip의 약어
* 칩 안에 시스템 전체가 통합되어 있는 반도체 칩
* 마이크로 프로세서, 메모리, 그래픽 처리 장치, 통신 인터페이스 등 포함
* 작은 크기의 효율적인 디지철 장치를 만들기 위해 사용
* 주로 모바일 기기, 임베디드 시스템, 스마트폰, 태블릿 컴퓨터 등에 적용
* 전체 시스템의 크기와 전력 소비를 줄이는 데 도움
* 고성능과 저전력 소비를 동시에 제공하는 장점


### 동작 원리
* 고밀도 집적 회로 기술을 사용, 다양한 HW 요소 통합
* 작은 공간에 많은 기능 포함, 각 구성 요소 간에 효율적인 통신을 가능하게 함
* CPU, GPU, 메모리 컨트롤러, 주변 장치 컨트롤러 등 다양한 구성 요소가 하나의 칩 안에 집적
* SoC의 핵심은 컴퓨터 아키텍처와 시스템 버스 인터페이스
* 컴퓨터 아키텍처 : CPU의 명령 집합과 동작 방식을 정의, SoC의 핵심 처리 기능 담당
* 시스템 버스 인터페이스 : SoC 내부의 다양한 구성 요소 간에 데이터 및 제어 신호 전달 역할
* 내부의 구성 요소들은 데이터 및 제어 신호를 주고 받으며 상호 작용
* CPU는 메모리로부터 데이터를 읽어와 연산 수행, GPU로 데이터를 전달, 그래픽 처리
* 이러한 구성 요소들은 시스템 버스를 통해 데이터를 주고 받으며, 필요에 따라 상호작용하며 작업 수행
* SoC는 전력 관리 기능을 내장, 전력 소모를 효율적으로 관리
* 사용하지 않는 기능을 비활성화하거나 전력 소모가 낮은 모드로 전환, 에너지 절약
* 배터리 수명 연장 혹은 장치의 발열 관리 기대 가능
* SoC는 SW 지원을 제공, 제조사는 SDK를 제공해 SoC의 기능을 활용 가능한 API 제공

### 주요 특징
1. 고성능, 저전력 소비
	고성능 처리와 동시에 저전력 소비를 가능하게 함, 이는 칩 내부의 다양한 HW 요소를 효율적으로 관리하고 제어함으로써 달성

2. 작은 크기
	여러 컴퓨터 HW요소를 하나의 칩에 통합, 전체 시스템의 크기 축약 가능
	이는 모바일 기기 혹은 임베디드 시스템 등 작은 크기의 제품 개발에 이점 제공

3. 빠른 개발 및 생산
	통합된 시스템을 제공하기 때문에 개발 및 생간 과정을 간소화 가능
	이는 제조사들이 제품을 빠르게 출시하고 시장에 대응하는 데 도움

### 구성 요소
1. 중앙 처리 장치(CPU)
	SoC의 핵심 요소, 데이터 처리 및 명령 실행 담당
	다양한 명령어 세트 지원
	코어 수에 따라 단일, 듀오, 쿼드 등 다양한 구성 가능

2. 그래픽 처리 장치(GPU)
	2D 및 3D 그래픽 처리를 담당
	게임, 비디오 재생, UI 애니메이션 등 시각적 요소를 처리해 부드러운 그래픽 퍼포먼스를 제공

3. 메모리
	시스템 메모리와 캐시 메모리 포함
	시스템 메모리는 프로그램 실행 및 데이터 저장에 사용
	캐시 메모리는 CPU에 가까워 데이터 액세스 속도 높힘

4. 디지털 신호 처리(DSP)
	음성 처리, 이미지 처리, 신호 처리 등과 같은 디지털 신호 처리 작업을 수행하는 특수한 프로세서
	오디오, 비디오 코덱, 음성 인식 등에 사용

5. 네트워크 인터페이스
	네트워크 연결을 지원하기 위한 인터페이스를 갖춤
	Wi-Fi, BLE, LTE, 5G 등의 무선 통신을 위한 모듈 내장 혹은 외부 인터페이스를 통해 연결

6. 주변 장치 인터페이스
	다양한 주변 장치와의 통신을 위한 인터페이스 제공
	USB, HDMI, 카메라, 센서 등과의 연결 지원
