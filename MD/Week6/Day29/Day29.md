## File / Directory
#### File System
* 파일 : 컴퓨터에서 데이터를 저장하고 조직화하는데 사용하는 기본 단위

* 포맷 : 저장 장치에 파일 시스템을 설치하는 것
#### Network Filesystem

* sysfs/procfs/tempfs등 여럿 사용

#### 파일의 종류
* 일반 파일(-) - 텍스트, 동영상, 사진 등
* 디렉터리(d) - 리눅스에선 디렉도 파일의 한 종류
* 심볼링 링크 파일(l) - 어떤 파일을 가리키는 파일
* 블록 디바이스 파일(b) - 블록 장치 제어 파일
* 문자 디바이스 파일(c) - 문자 장치 제어 파일
* 파이프 파일(p) - 프로세스 간 통신에 사용되는 파이프를 나타내는 파일
* 소켓 파일(s) - 프로세스 간 통신에 사용되는 소켓을 나타내는 파일
=> 디바이스도 파일의 일종, block -> 블록 단위로 데이터 전송, 버커 캐시 거침
#### 루트 디렉터리
* 디렉터리 : 파일 시스템을 계층화 할 때 사용하는 도구
* 루트 디렉터리 : 파일 시스템의 최상단에 위치하는 디렉터리
	* /bin : 기본 실행 명령어
	* /dev : 디바이스 파일
	* /etc : 시스템 설정 파일
	* /home : 사용자 홈 디렉
	* /lib : 시스템 라이브러리
	* /mnt : 디스크 등 임시 마운트 시 사용
	* /proc : procfs 마운트
	* /root : root 홈 디렉
	* /sbin : 시스템 관리용 명령어
	* /sys : sysfs 마운트
	* /tmp : 시스템 임시 파일
	* /usr : 시스템 소프트웨어

## 사용자와 사용자 그룹
#### 사용자 종류
* root 사용자(SA user) : 리눅스 시스템의 모든 권한을 가진 관리자 계정
	* 권고
		* root 사용자의 암호 복잡화, 주기전 변경
		* 시스템 외부에서 root 사용자로 로그인 못하게 설정
	* root 사용자 권한으로 명령 실행
		* su 명령어 사용
		* $su option - user
	* sudo 명령어 사용
		* $sudo option command
	* runuser 명령어 사용
		* runuser option -u user command
		* runuser option - user command
		* -l - 타 사용자 쉘 시작
		* -g roup - 타 사용자 기본 그룹 지정
		* -s shell - 타 사용자 기본 쉘 지정
		* -c command - 타 사용자로 실행 할 명령어 지정
		* -P - 타 사용자의 프로세스 ID 지정
		* -f - 타 사용자 환경변수 지정
* 시스템 사용자 : 리눅스 시스템에서 만든 사용자
* 일반 사용자 : root 사용자와 시스템 사용자를 제외한 모든 사용자
* 사용자 정보 관리하는 /etc/passwd 파일
	* gilbut:x:1000:1000:gilbut:/home/gilbut:/bin/bash
	* username:passwd:userID:GroupID:추가설명:homeDir:LoginSHell
	* 사용자 이름 : 영문자, 숫자, 밑줄, 마침표로 구성
	* 암호 : 필드에 x표기, 실 암호는 암호화된 형태로 /etc/shadow 파일에 저장
	* 사용자 ID : 0보다 크거나 같은 정수로 설정, 크기는 의미 x
	* 사용자 그룹 ID : 0보다 크거나 같은 정수로 설정
	* 추가 설명 : 해당 사용자 설명 가능한 내용 작성
	* 홈 디렉 : /home 디렉 밑에 사용자 이름과 같은 디렉 설정
	* 로그인 쉘 : 쉘 실행 바이너리 위치 절대 경로로 표시
* 사용자 그룹 정보를 관리하는 /etc/group 파일
	* adm:x:4:syslog,gilbut
	* UserGName:passwd:UserGID:UserList
	* 사용자 그룹 이름 : 최대 32자, 영문, 숫자, 하이픈, 언더스코어 구성
	* 비밀번호 : x로 표시
	* 사용자 그룹 : 0보다 크거나 같은 정수로 설정
	* 사용자 목록 : 사용자 목록에 다수 사용자가 있는 경우 쉼표로 구분
* 사용자 추가 및 삭제
	* $adduser option user_name
		* --home dir : 사용자 홈 디렉을 지정한 디렉으로 설정
		* --shell shell : 사용자의 로그인 쉘을 지정한 쉘로 설정
		* --ingroup group_name : 사용자를 지정한 사용자 그룹에 포함

## 소유권과 권한
#### 파일 소유권
* 소유권 : 파일이 사용자의 소유임을 나타내는 속성
* drwxr-xr-x 2 gilbut gilbut 4096 May 21 12:49 Desktop
	* 파일 소유자 / 파일 소유 그룹
* chown 명령어 - 파일의 소유권 변경
	* $chown option user\[:group] file
		* -R : 디렉 및 하위 소유권 모두 변경
		* -v : 변경된 파일과 디렉 정보 출력
		* -c : 실제 변경된 항목만 출력
#### 파일 권한 종류
* 파일 권한 : 해당 파일에 어떤 행위를 할 수 있는지를 정의 한 것
* 파일 권한을 파일의 소유자, 소유 그룹, 일반 사용자 별로 따로 설정
* 의미 표기법 사용 : 현재 파일에 설정된 권한에 어떤 변화를 줄지 정하는 방식
* rwt 권한 : read, write, stickyBit
	* stickyBit -> 타 사용자에게 UID 대여
