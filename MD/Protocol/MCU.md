### MCU란?
Microcontroller Unit의 약자, 작은 규모의 컴퓨터 시스템을 하나의 칩 안에 통합한 장치
CPU, 메모리(RAM, ROM), IO 포트, 타이머, 컴파일러 등 다양한 주변 장치 통합, 제공
소형/저전력 시스템에서 사용

### 주요 특징
작고 저전력이며 다양한 주변 장치와의 상호 작용이 필요한 시스템에 적합
경제적이고 효율적인 솔루션 제공

1. 통합된 아키텍쳐
	MCU는 CPU, 메모리(RAM, ROM), IO 포트, 타이머, 컴파일러 등 다양한 주변장치를 하나의 칩에 통합해 제공, 이는 MCU를 소형, 저전력 시스템에 이상적으로 만듦
	
2. 저전력
	MCU는 저전력 시스템에 사용되는 것이 일반적
	이는 MCU가 배터리로 작동하는 장치나 에너지 효율이 중요한 장치에 이상적이라는 것을 의미
	MCU는 전력 소모를 최소화하는 기능을 포함, 저전력 모드로 전환해 에너지 절약 가능
	
3. 입출력 인터페이스
	다양한 입출력 인터페이스 제공
	디지털 입출력 포트, 아날로그 입력 포트, UART, SPI, I2C 등의 인터페이스를 통해 주변 장치와 통신 가능
	이러한 입출력 인터페이스는 MCU를 타 장치와 상호 작용하는 데 사용 가능케 함
	
4. 작은 크기와 저렴한 가격
	작은 크기의 칩으로 제공, 저렴한 가격으로 구매 가능
	MCU를 대량 생산 및 저비용 임베디드 시스템에 이상적으로 만듦
	
5. 실시간 처리
	실시간 처리 요구 사항을 충족하는 능력 가짐
	실시간 시스템은 정확한 시간 제약을 가지며, 외부 이벤트에 신속하게 응답해야 함
	MCU는 내장된 타이머와 인터럽트 기능을 사용하여 실시간 시스템 요구 사항 처리 가능
	
6. 프로그래밍 가능성
	프로그래밍 가능한 장치, 다양한 SW 애플리케이션 실행 가능
	MCU는 주로 C 또는 어셈블리 언어 사용해 프로그래밍
	개발자는 MCU에 필요한 기능을 구현하기 위해 주어진 SW 작성 가능
	또한, MCU는 주변 장치와의 상호 작용을 위해 다양한 프로토콜 및 통신 인터페이스 지원

### 주요 종류

1. ARM 기반 MCU
	* 널리 사용되는 MCU, 다양한 제조업체에서 ARM 아키텍처를 기반으로 한 MCU 제공
	* ARM Cortex-M 시리즈는 저전력 소비와 뛰어난 성능 결합, 다양한 응용 분야에 적합
	* ARM 아키텍처의 유연성으로 인해 다양한 기능과 통신 인터페이스를 제공하는 MCU 존재
	* 주요 업체 : STMicroelectronics, NXP, Texas Instruments 등

2. AVR 기반 MCU
	* Atmel 사에서 개발한 ACR 아키텍처를 기반으로 한 MCU로 알려짐
	* 작고 저전력, 비교적 간단한 응용 분야에 적합
	* Arduino와의 호환성으로 널리 사용되는 특징
	* 다양한 주변 장치 및 통신 인터페이스 제공
	* 주요 제조업체 : Microchip

3. PIC 기반 MCU
	* Microchip사에서 개발한 8비트 및 16비트 PIC MCU가 널리 사용
	* 저렴, 저전력, 다양한 주변 장치와의 상호 작용이 필요한 임베디드 시스템에 많이 사용
	* 풍부한 기능, 다양한 통신 인터페이스 제공
	* 주요 제조업체 : Microchip

4. MSP 기반 MCU
	* Texas Instruments 사의 MSP MCU는 저전력 시스템에서 사용되는 것이 일반적
	* 에너지 효율성 우수, 저전력 모드와 전력 관리 기능 갖춤
	* 다양한 통신 기능 지원, 센서 네트워크 및 무선 통신 애플리케이션에 적합
	* 주요 제조업체 : Texas Instruments

5. TriCore 기반 MCU
	* TriCore는 인피니온에서 개발한 자체 아키텍처로, 주로 자동차 전자 시스템에서 사용되는 MCU
	* TriCore MCU는 고성능 및 안정성이 요구되는 애플리케이션에 적합
	* 자동차 시스템 등과 같이 높은 처리 요구 사항을 가지는 애플리케이션에서 강력한 성능 발휘
	* 주요 제조업체 : Infineon
