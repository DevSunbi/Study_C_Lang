### IMU의 구성 요소
1. 가속도계
	* 측정 값 : 직선 운동에 따른 가속도 및 중력 가속도
	* 역할 : 물체가 어느 방향으로 움직이는지, 지면을 기준으로 얼마나 기울어져 있는지 (x, y, z, roll, pitch)

2. 자이로스코프
	* 측정 값 : 각속도(단위 시간 당 회전하는 각도)
	* 역할 : 물체가 얼마나 빠르게 회전하고 있는지 측정. 이를 시간으로 적분하면 현재의 각도(Yaw, Roll, Pitch)를 구할 수 있음
	* Roll, Pitch : 움직임이 없을 때 정확하지만, 움직일 땐 자이로스코프 필요
		가속도계는 중력 가속도를 이용, 정적인 상태에서 정확
		주행 중 발생하는 진동이나 가속 성분에 취약, 값이 심하게 튐
		자이로스코프는 동적인 움직임에 빠르게 반응, 노이즈에 강함
		적분 오차로 인한 드리프트 발생
		따라서 두 센서의 데이터를 융합, 가속도계로 장기적인 절대각 보정, 자이로스코프로 단기적인 신뢰성 확보를 통해 정확한 Roll/Pitch 산출

3. 지자기 센서 
	역할 : 지구의 자기장 감지, 나침반처럼 절대적인 북쪽(Heading) 기준 잡을 수 있게 도움
	* 수평 상태에서 어느 방향을 보고 있는지 모름(중력으로 위/아래는 앎)
	* 자이로스코프의 오차 누적으로 인해 옆을 보고 있다 착각하는 현상을 지자기 센서를 통해 기준을 바로잡음
	* 실내나 도심 주행 시 지자기 센서의 신뢰도 문제
		* 확잘 칼만 필터 등 센서 퓨전 알고리즘을 사용해 왜곡이 심한 구간에서 지자기 센서의 가중치를 낮추고 자이로스코프나 휠 속도 센서 데이터의 비중을 높혀 전체적인 신뢰성 유지

### 노이즈와 드리프트
1. 가속도계의 한계 : 진동에 매우 민감, 값이 요동침(High-Frequency noise)
2. 자이로스코프의 한계 : 시간이 지날수록 오차 누적, 각도가 조금씩 틀어지는 드리프트 현상 발생
* 해결법(필터 알고리즘) : 가속도계의 장점과 자이로스코프의 장점을 결합 -> 상보 필터나 칼만 필터를 사용해 데이터의 신뢰도 증진
* 가속도 데이터 적분해 속/거 구하면 오차 발생 해결 방안
	* IMU데이터 의존이 아닌 차량 바퀴 속도 센서 데이터 / GPS 위치 정보를 결합하는 센서 퓨전 기법을 활용, 오차를 주기적으로 보정
